## Bundles
> Basic Ubuntu and Debian Linux Easy Setup Script

Simple copy and paste the following command to use the script

```
git clone https://github.com/MingFei2001/bash-script.git
cd bash-script
sudo bash bundles.sh
```

Option 1 will only install some basic CLI software needed in daily life\
Option 2 will install additional software for content creation and consumption tools\
Option 3 will combine both to quickly setup a home desktop that contains most software that i personally used

Edit the script to remove or adjust the software that you want to use.
